# Attacchi di Phishing: Frequenze, Vittime e Spese
Progetto di Visualizzazione Scientifica sul Phishing
Questo è un progetto per il corso di "Visualizzazione Scientifica" nel quale ci occupiamo di visualizzare ed analizzare dati relativi agli attacchi informatici di Phishing:

- In primis osserviamo nel panorama degli attacchi informatici dove si posiziona, in quanto a frequenza, questo tipo di attacco.
- Poi ci siamo occupati di vedere nello specifico, dal 2014 al 2024, i siti soggetti ad attacchi di Phishing e le relative campagne mail, oltre che la media mensile dei Brand attaccati.
- Successivamente abbiamo deciso di osservare le spese causate alle aziende, calcolate tenendo in considerazione Rilevamento ed escalation, notifica, risposta post-breach e affari persi.
- Infine poniamo attenzione alle vittime per comprendere se età,genere,istruzione possono essere predittori principali di vulnerabilità a questo tipo di attacco
